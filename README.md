# Portafolio Personal – Juan Valencia

Este es mi portafolio personal como desarrollador de software en formación. El proyecto consiste en un sitio web donde presento mi perfil profesional, habilidades técnicas, proyectos realizados y un video de presentación. Este portafolio fue desarrollado como parte de mi proceso de aprendizaje en desarrollo web frontend.

---

## Descripción del proyecto

El objetivo de este proyecto es crear un sitio web profesional que funcione como portafolio personal, permitiendo mostrar mis conocimientos en desarrollo web utilizando tecnologías como HTML, CSS y JavaScript.

Este sitio incluye secciones de presentación personal, galería de proyectos, habilidades y contenido multimedia.

---

## Características

- Diseño web moderno
- Interfaz responsiva adaptable a diferentes dispositivos
- Sección de presentación personal
- Galería de proyectos
- Integración de video de presentación
- Organización estructurada del código
- Navegación clara y sencilla

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
